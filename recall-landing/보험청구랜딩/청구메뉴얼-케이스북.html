CHAPTER 1 · 잇몸 · 치석제거

1 “잇몸에서 피나요” — 연 1회 치석제거▶

🎬 상황40대 환자. “양치할 때 피가 나요, 스케일링 한 번 받고 싶어요.” 만 19세 이상, 올해 아직 스케일링 안 받음.
✅ 정석대로

1년에 1번 되는 연1회 치석제거(치석제거-나, 전악). 진단 근거를 챠팅에 남기는 게 핵심.

상병 K05.08 기타 급성치은염K05.10 단순변연부 만성치은염K05.22 급성치관주위염K05.30 만성단순치주염

챠트 그대로c.c) 양치 시 잇몸 출혈 O) 전반적 치은염, #36 37 특히 염증성 출혈 심함, 치석 침착(+) A) 만성 치은염 P) 전악 연1회 치석제거 필요 → 요양기관정보마당 등록함 → 연1회 치석제거 시행함
  • 요양기관정보마당(바우처) 등록 필수 — 이거 빠지면 나중에 조정.
  • 치아 14개 이하여도 전악 1회로 시행.
🗣️ 환자에게 이렇게
“지금 잇몸에 염증이 있어서 피가 나는 거예요. 스케일링은 1년에 한 번 건강보험 적용돼서 부담 크지 않으세요. 오늘 전체 한 번 깨끗이 하겠습니다.”
❌ 이렇게 하면 삭감
  • O·A 없이 “스케일링 시행”만 딸랑 → 급여 근거 없음, 조정.
  • 상병을 K05.31 만성복합치주염으로 → 연1회엔 적용착오(복합은 치주치료용 상병).
  • 치식을 물결(~)로 뭉뚱그림 → missing 치아 포함해 정확히 안 적으면 안 됨.
  • 연1회 스케일링 하고 6개월 안에 또 전악 스케일링 → 재시행 불가.
연1회 스케일링도 “왜 필요했는지(출혈·치석·염증)”를 O·A에 남겨야 급여. 상병은 치은염~단순치주염, 복합치주염은 금지.

2 “잇몸치료 받으세요” — 치주용 스케일링(전처치)▶

🎬 상황치주염이 꽤 있는 환자. 스케일링 한 번으로 안 끝나고 잇몸치료(치근활택·소파)까지 이어질 예정.
✅ 정석대로

연1회(치석제거-나)가 아니라 치주치료 전처치용 치석제거(치석제거-가, 1/3악당). 잇몸치료로 이어진다는 걸 남긴다.

상병 K05.30 만성단순치주염K05.31 만성복합치주염K05.4 치주증 (주로 만성)

챠트 그대로잇몸치료 예정 / 잇몸치료 위한 전처치(치석제거) 시행함
💰 여기서 더 받기
  • 같은 날 치주낭측정검사(1/3악당)를 함께 — mm로 치아당 2면 이상 기록하면 정당 청구.
  • 전처치 스케일링 후 다음 내원에 치근활택술 → 치주소파술로 단계 상승(케이스 3 참고). 한 번에 끝내지 말고 병행·단계로 가는 게 자연스럽고 객단가도 올라감.
🗣️ 환자에게 이렇게
“스케일링만으로는 잇몸 속 염증이 안 잡혀요. 오늘 겉을 정리하고, 다음에 잇몸 속까지 치료하겠습니다. 다 보험 적용됩니다.”
❌ 이렇게 하면 삭감
  • 치주용(-가)으로 하고 그날로 끝(치료 종결) → 사실상 예방 스케일링이라 비급여 처리(연1회 소진 개념).
  • 같은 달에 잇몸치료 못 이어감 → 내역설명 없으면 왜 스케일링만 했냐고 잘림.
  • 환자가 안 옴 → 아무 기록 없이 두지 말고 아래처럼:
환자분 사정으로 잇몸치료 내원하지 않으심 / 연락드렸으나 추후 오신다고 함
치주용 스케일링은 “잇몸치료로 이어진다”가 전제. 그날 끝나면 비급여, 못 이어가면 내역설명.

3 잇몸치료 단계 밟기 — SC → RP → 소파술(재시행 규칙)▶

🎬 상황만성 치주염 환자. 스케일링(SC) 후 치근활택술(RP), 더 깊으면 치주소파술(CU)까지. 이걸 “언제 다시 하면 얼마 받나”가 헷갈림.
✅ 정석대로 — 재시행 기간표(이것만 외우면 됨)
행위후처치로만(≈0)50% 산정100% 산정
치석제거 SC3개월 이내3~6개월6개월 초과
치근활택술 RP1개월 이내1~3개월3개월 초과
치주소파술 CU1개월 이내1~3개월3개월 초과
치은박리소파술 Flap—6개월 이내6개월 초과

같은 행위를 다시 하면, 기간에 따라 100%→50%→후처치로 깎여서라도 청구는 됨.

💰 여기서 더 받기 — 단계를 올려라
  • 치주소파술은 마취 동시청구가 필수이자 정당 항목 → 마취 앰플까지 함께.
  • 낮은 행위(SC)에 머물지 말고, 실제 상태가 되면 RP → 소파술로 단계 상승. 상위 행위는 100% 새로 인정.
❌ 이렇게 하면 삭감
  • 역방향 금지 — 소파술 하고 나서 다음에 치근활택술로 내려가면 청구 불가.
  • 치주소파술을 급성 상병으로 → 만성 상병만 인정.
  • 전처치(SC) 없이 바로 소파술 → 조정. (초진에 바로 소파술 청구하면 치근활택술로 깎임)
  • 마취 없이 소파술 → 삭감.
타기관에서 전처치했을 때 내역설명타 치과에서 치석제거 후 내원하여 소파술 시행함
잇몸치료는 위로만 올라간다(SC→RP→CU→Flap). 소파술=만성+마취+전처치, 이 3종 세트.

  • NiTi랑 hand file 둘 다 재료 청구 → 하나만.
엔도는 “발수1·확대2·세척5·근충1”이 상한. 치수염엔 항생제 금지, 약 주려면 농양 상병.

5 뿌리 염증이 심해 소독을 많이 함 — 세척 5회 초과▶

🎬 상황근단 농양이 심해 근관세척을 6~7번 해야 가라앉음.
✅ 정석대로

근관세척은 원칙 5회 이내지만, 농양은 5회 초과도 인정 — 단 왜 많았는지 내역설명.

내역설명근단 염증 심하여 근관세척 다수 시행함 (통증 호소 · 지속적 배농, 5회 이상)
❌ 이렇게 하면 삭감
  • 내역설명 없이 6회+ → 초과분 조정.
  • 세척과 발수/가압충전을 같은 날 → 동시 불가.
  • saline(세척액) 재료대 따로 청구 → 불가.
세척 5회 넘으면 “농양·배농”을 내역설명으로. 세척은 발수·근충과 같은 날 못 붙인다.

6 예전에 신경치료한 이가 또 아픔 — 재근관치료(Re-endo)▶

🎬 상황몇 년 전 신경치료 한 #26이 다시 아프고 뿌리 끝에 병소. 기존 충전물 제거하고 다시 함.
✅ 정석대로

상병 만성근단치주염근단주위육아종근단주위농양치근낭

  • 시작은 근관내 기존충전물 제거부터. 그 다음 단계는 근관세척부터 이어감.
  • 발수·근관와동형성은 청구 X(이미 예전에 했으니까).
❌ 이렇게 하면 삭감
  • 재근관치료인데 발수·와동형성 또 청구 → 삭감(중복).
  • 보철물 제거 + 근관내 기존충전물 제거 동시 → 100:50(둘 다 100 아님).
Re-endo는 “기존충전물 제거 → 세척”부터. 발수·와동형성은 없다.

사진(PA) + 내역설명 둘 다 없으면 청구 자체가 안 됨.

💰 여기서 더 받기
  • 난발치는 BURR 청구 가능(단 하루 1개만 인정).
  • 난발치 + 치조골성형술 동시 → 100:50 (상병 구분해서).
❌ 이렇게 하면 삭감
  • 사진·내역 없이 난발치 → 단순발치로 조정.
  • 잔존치근을 난발치로 → 잔존치근은 단순발치.
  • BURR 하루 2개 → 1개만.
난발치 = “분리삭제” 문구 + PA + 강직/유착/비대/만곡 상병. 이 셋 없으면 단순발치로 깎인다.

9 사랑니 매복 3종 — 단순 / 복잡 / 완전▶

🎬 상황#38 사랑니. 얼마나 묻혀 있고 어떻게 뽑았느냐로 수가가 완전히 달라짐. 챠팅 문구 = 청구 등급.
✅ 정석대로 — 문구가 곧 등급
등급챠팅에 그대로 쓸 정의비고
단순매복피막(잇몸) 절개하여 발치함사진 필수 · Silk 사용불가
복잡매복치아 분리삭제하여 발치함BURR 가능
완전매복골삭제 + 치아분리, 치관 2/3↑ 매복BURR 가능

상병은 매복 상병(상·하악 제3대구치 매복). 사진 필수.

💰 여기서 더 받기
  • 완전매복이 하치조신경과 겹치면 CT 인정 — 단 발치까지 이어진 경우만.
  • 실제 골삭제·분리했으면 등급 그대로(완전) 청구. 한 만큼 받는 것.
❌ 이렇게 하면 삭감
  • 사진 없는 매복치 → 단순발치·난발치로 심사조정.
  • 매복인데 상아질우식 상병으로 → 매복 상병 아니라서 조정.
  • 교정 목적 발치 → 비급여. (단 사랑니·충치 원인 동반하면 급여)
  • 실제로 골삭제 안 했는데 완전매복으로 → 현지조사 위험(허위).
매복은 “절개=단순 / 분리삭제=복잡 / 골삭제+분리=완전”. 사진 없으면 등급 다 깎인다.

10 잇몸이 곪아 부었어요 — 구강내소염술(I&D)▶

🎬 상황잇몸에 고름집. 칼(blade)로 째서 배농.
✅ 정석대로

상병 동 없는 근단농양동 없는 치주기원 농양

챠팅에 블레이드 사용·절개 행위와 사진이 있어야 함.

사진 없이 할 때 내역설명기존 엑스레이 참조하여 구강내소염술 시행함 (blade 절개 배농)
💰 동시청구 규칙(외워두기)
  • 발수 + 구강내소염술 → 발수100 · 소염술50
  • 구강내소염술 + 응급근관처치 → 소염술100 · 응급50
  • 구강내소염술 + 치석제거 → 소염술100 · 치석50
  • 구강내소염술 + 발치 → 발치만(소염술 X)
❌ 이렇게 하면 삭감
  • 사진·절개 행위 챠팅 없이 소염술 → 근거 없음.
  • 발치랑 같이 하고 소염술도 따로 청구 → 발치만 됨.
소염술은 “blade 절개 + 사진”. 발치와 같이 하면 소염술은 사라진다(발치만).

11 엑스레이 크게 찍어요 — 파노라마 · CT▶

🎬 상황여러 군데 문제가 있어 파노라마 한 장. 판독소견을 안 쓰면 잘림.
✅ 정석대로 — 산정되는 경우
  • PA만으로 부족 + 인접하지 않은 3곳 이상 원인치아
  • 매복치 · 턱관절(TMD) · 개구제한 · 장애로 구내촬영 곤란 (한 부위여도 가능)

판독소견 예시#36 37 치주염 확인함 / #46 우식 확인함 / #26 근단농양 확인함 #48 치근과 하치조관이 2mm 떨어져 있음
💰 CT까지 가는 정당한 경우
  • 신경치료 지속 동통 / 치근단절제술 해부학 확인 / 매복치가 하치조신경과 겹침 / 3치관 크기↑ 치근낭
  • CT는 판독소견서 작성·비치 필수.
  • 치근단 + 파노라마 둘 다 인정(치근단 촬영목적 내역설명 달면).
❌ 이렇게 하면 삭감
  • 파노라마 찍고 전악 클릭 청구 → 개연성 낮음. 판독소견에 적힌 부위만 클릭.
  • 판독소견 안 씀 → 판독료 조정.
  • 매복발치 안 했는데 매복 CT만 → 발치까지 이어진 경우만 인정.
파노라마·CT는 “판독소견 쓴 부위만” 청구. 소견 없으면 없는 검사다.

12 보기만 하고 감 — 검진·상담만(기본진찰료)▶

🎬 상황“어디 아픈 건 아니고 봐주세요” / 상담만 하고 감. 기본진찰료만 청구는 가장 위험한 청구.
✅ 정석대로

기본진찰료만 청구는 꼭 필요할 때 + 상병명 다양 + 내역설명일 때만. 진찰료만 되는 경우: 상담·검사만, 간단한 구강연조직 처치, 구내염 약물도포, 치수냉온검사, 발치 전 동통감소 처치, 처방전만 발행, 발치 미완 중단.

챠트/내역설명충치로 원장님 진단 받음 (검진) 치수 냉온검사 시행 / 발치 전 동통감소 위해 드레싱
❌ 이렇게 하면 삭감·조사
  • 검진 O.E / CHECK / N/S만 딸랑 → 급여·비급여 구분 불가. 실제 구강상태를 적어야.
  • 기본진찰료만 연달아 2~3회 → 일률 진료로 파악, 사유 필요.
  • 미완성 진료를 초진으로 → 삭감(끝나지 않았으면 내역설명).
기본진찰료만은 “왜 진료 없이 진찰만인지”를 반드시 남긴다. 검진은 O.E만 쓰지 말고 실제 소견을.

13 약만 타러 왔어요 — 처방전·항생제 함정▶

🎬 상황“잇몸이 붓는데 약만 주세요” / 가족이 대신 약 타러 옴.
✅ 정석대로
  • 본인 내원 처방전만 → 기본진찰료 산정.
  • 가족이 대신 내원 처방 → 진찰료 50%.
  • 발급기간 내 재발행 → 진찰료 산정 불가 / 기간 지난 뒤 → 산정.
챠트잇몸치료 진단 났으나 환자분이 약처방만 원하심 (진찰·상담) 처방전만 발행함
❌ 이렇게 하면 삭감
  • 치수염 상병에 항생제 → 불가. 항생제는 농양·발치·수술 상병에.
  • 비가역·가역 치수염, 우식, 치석제거, 기본진료에 항생제 → 상습처방으로 잡힘.
  • 위장약 나가는데 위장질환 근거 없음 → 근거 챠팅 필요(예: R)속쓰림).
  • 비급여 진료인데 보험 약처방 → 처방도 비급여(기타)로.
  • 인사돌·오라메디·케어가글 → 비급여(2012.3~). 헥사메딘 가글은 치료기간 중 1회 100ml만 급여(단독 불가).
치수염·우식·스케일링엔 항생제 금지. 약엔 항상 “그 약이 필요한 상병·근거”가 챠트에 있어야.

19 마취를 여러 번 했는데 — 마취 동시청구▶

🎬 상황같은 부위에 침윤마취 + 전달마취를 같이 함.

  • 도포마취는 불인정.
  • 앰플은 실제 사용 개수만큼 청구, 성분·상품명 기입.

20 비급여 하러 왔는데 보험도 같이 — 정당한 1+1▶

🎬 상황크라운·임플란트 등 비급여로 온 환자. 그 김에 보이는 보험 질환을 병행. 없는 걸 만드는 게 아니라, 실제 있는 걸 놓치지 않는 것.
💰 개연성 있게 붙이는 정당한 조합
  • 비급여 방문이라도 질병 확인되면 보험 병행 — CA 확인 → 지각과민/보험 충전.
  • 프렙·인상 전 부분 치석제거 / 크라운·임플란트 셋팅 시 주변치아 스케일링(증상 기입).
  • 소아: 치면세마·실란트 권유, 진정처치·치수복조.
  • 발수·발치·치주 부위는 파노라마 유무 상관없이 치근단 필수(찍고 청구).
🗣️ 환자에게 이렇게
“이 쪽 크라운 하는 김에 보니까 옆니 잇몸에 염증이 있어요. 오늘 같이 정리해드릴게요, 이건 보험 돼요.”
❌ 넘으면 안 되는 선
  • 증상·소견 없이 스케일링을 습관적으로 붙이기 → 개연성 없어 조정.
  • 비급여 진료를 급여로 둔갑 → 부당청구(비급여 후 급여청구는 7대 유형).
  • 안 한 행위를 붙이기 → 허위, 현지조사.
1+1은 “실제 있는 질환 + 증상 기입 + 사진”. 증상 없이 습관적으로 붙이면 그게 삭감·조사 사유.
CHAPTER 7 · 급여 틀니 · 임플란트

21 65세, 틀니 하고 싶어요 — 급여 틀니▶

🎬 상황만 66세 환자. 완전틀니(무치악) 또는 부분틀니.
✅ 정석대로
  • 대상 만 65세 이상 · 1악당 7년에 1회 · 5단계로 진행 · 사전등록제.
  • 본인부담 총액 50%(의료급여 1종 20·2종 30).
  • 무상보상 3개월 최대 6회(재진 진찰료만).
  • 유지관리: 첨상·개상 연1회 / 조직조정·수리·조정 연2회 / 교합조정 단순 연4회 — 등록 후 시행.
❌ 이렇게 하면 삭감
  • 사전등록 없이 진행 → 조정.
  • 유지관리 횟수 초과 / 등록 전 시행 → 불가.
틀니는 65세↑·7년1회·5단계·사전등록·본인 50%. 유지관리는 등록 후 횟수 내에서만.

22 65세, 임플란트 보험 되나요 — 급여 임플란트▶

🎬 상황만 67세, 어금니 하나 빠진 부분무치악. 보험 임플란트 문의.
✅ 정석대로
  • 대상 만 65세↑ 부분무치악 · 평생 2개 · 전·구치부 · 1·2·3단계 · PFM · 상병 K08.1.
  • 본인부담 50%(의급 1종20·2종30). 중단 시 해당 단계까지만.
3단계 셋팅일(#26)K08.1 / ST-abut adap / straight(or angle) abut adap PFM setting / T.B.I
❌ 비급여인 것 (급여로 청구하면 부당)
  • 완전무치악, 관골 식립, 일체형, PFM 외 보철, 커스텀 어버트먼트 재료대, 상악동거상·골이식 → 비급여.
  • 평생 2개 초과 → 불가.

재수술: 2단계 고정체 제거 후 공단 시술중지 신청 → 재등록 → 2단계 행위료 50% + 재료대 100%(3단계 전만).

보험 임플란트=65세↑·부분무치악·평생2개·PFM·K08.1·본인50%. 골이식·상악동거상·완전무치악은 비급여.

23 얼마 내면 돼요? — 본인부담·수납▶

🎬 상황데스크에서 “보험 얼마예요?” 물음. 나이·금액대로 다름.
✅ 정석대로 — 치과의원 본인부담
구분6세 미만6~65세 미만65세 이상
본인부담율21%30%총액 15,000원↓면 1,500원 / 초과면 30%

종별가산: 치과의원 건보 15%·의료급여 11% / 치과병원 20%·15%. 소아·노인·장애인 가산 별도.

💰 미수 안 생기게 — 수납 순서
  • 보험진료비 먼저 차감 → 보험은 그때그때 별도 청구·수납이 최선.
  • 보험금액은 할인 불가(할인하면 본인부담 과소 → 부당).
65세↑는 15,000원 넘느냐로 정액(1,500)/정률(30%) 갈림. 보험비는 할인 금지.
CHAPTER 8 · 삭감당했을 때

  • 기한 90일 이내(가급적 60일 넘기지 말 것).
  • 공단 등록 미비로 조정된 건은 진료일로 등록 후 재심사.
❌ 애초에 잘리는 근본 원인 3가지
  • O·A(객관·진단) 없이 P(행위)만 → SOAP 안 지킨 챠팅이 삭감 1순위.
  • 사진·내역설명 누락(난발치·매복·소염술·파노라마).
  • 챠팅한 마취·행위와 청구 불일치.
삭감은 끝이 아니라 “보완 후 90일 내 재청구”. 근본예방은 SOAP·사진·내역설명·마취일치 4개.
